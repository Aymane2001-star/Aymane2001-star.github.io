# Aymane Lakouiss Portfolio

Static portfolio website for Aymane Lakouiss, research engineer in deep learning and computer vision.

## Local preview

Open `index.html` in a browser, or serve the folder locally with:

```bash
python -m http.server 8012
```

Then visit `http://localhost:8012/`.

## Deployment

This project is ready for GitHub Pages. Publish the repository from the `main` branch and the root folder.
